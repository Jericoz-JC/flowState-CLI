# flowstate-cli

A unified terminal productivity system for notes, todos, and focus sessions with semantic search.

## Installation

```bash
npm install -g flowstate-cli
```

## Usage

```bash
flowstate
```

## Features

- **Notes**: Quick capture, tagging, and organization
- **Todos**: Task management with priorities and due dates
- **Focus Sessions**: Pomodoro-style timer with session tracking
- **Semantic Search**: Local ONNX-powered semantic search
- **Linking System**: Connect notes and todos through relationships

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Ctrl+X` | Quick capture note |
| `Ctrl+N` | Notes screen |
| `Ctrl+T` | Todos screen |
| `Ctrl+F` | Focus session screen |
| `Ctrl+/` | Semantic search |
| `q` | Quit |
